# Itihas

**Small actions. Big ripples. A better world.**

Itihas is an open-source platform born from a simple belief: that young people don't need to wait to change the world — they can start right where they live. Founded by **Sahiti Mulagada**, Itihas began as a personal project to organize community initiatives in her residential neighborhood and has since grown into a collaborative movement inviting like-minded people everywhere to contribute.

> *"Itihas"* means *history* — because every small act of kindness, every micro-change, becomes part of a larger story we write together.

---

## The Story

It started in **Rainbow Vista Rock Gardens**, a residential community in Visakhapatnam, India. Sahiti saw a gap — kids had energy and ideas but no platform to channel them. So she built one. What began as organizing a single entrepreneurship event for kids (*BizKids*) evolved into a full digital platform supporting donation drives, environmental initiatives, mentorship programs, and community collaboration.

In **April 2025**, Itihas v1.0 went live — a fully functional web application designed, developed, and deployed by a high school student. The platform served real users, managed real events, and handled real community engagement from day one.

Now, Itihas is **open source** — because the best ideas grow when shared.

---

## What Itihas Does

### BizKids — Youth Entrepreneurship Events
A complete event platform where kids set up stalls, sell products, and learn real-world business skills — all organized by young people, for young people, on a non-profit basis.
- Stall registration and management
- Live search across stalls by name, school, or category
- Review and rating system with Google OAuth
- Real-time leaderboard
- Full financial transparency — every rupee accounted for

### Collaboration Hub
A space for community members to share ideas, post updates, and organize around causes that matter.
- Rich text post creation
- Event categories: Public Events, Gramasabha, LPS Events, and more
- Comment threads for open discussion

### Projects Showcase
Highlighting youth-led initiatives — from urban labs to sports programs — with dedicated pages for each project's story and impact.

### Blog & Member Profiles
Sharing updates, member stories, and the journey of the organization as it grows.

## Version History

### v1.0 — April 2025
The first public release. Built and launched as a personal project by Sahiti Mulagada while in high school.

- **BizKids 2025 Event Management** — End-to-end platform for the flagship youth entrepreneurship event: stall registration, live search, peer reviews with star ratings, real-time leaderboard, photo gallery, and full financial transparency
- **Collaborative Network** — A community hub where members post updates, organize events, discuss ideas through comment threads, and coordinate across categories like Public Events, Gramasabha, and LPS Events
- Full-stack web application with Next.js 15 and React 19
- Google OAuth authentication for trusted community participation
- Blog system and member profiles to tell each contributor's story
- Responsive, mobile-first design accessible on any device
- Static site export for fast, reliable hosting
- Custom Node.js server for production deployment

**This version proved that a student-built platform could power a real community event — managing BizKids 2025 registrations, collecting live user reviews, and connecting a collaborative network of young changemakers in production.**

### v2.0 — Open Source (Current)
Itihas is now open to contributors worldwide. The vision: a global network of young people making micro-changes in their own communities, inspired by each other's work.

## Getting Started

### Prerequisites
- Node.js 18+
- npm, yarn, pnpm, or bun

### Run Locally

```bash
# Clone the repository
git clone https://github.com/your-username/itihas.git
cd itihas

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

### Build for Production

```bash
npm run build
node server.js
```

The production server runs on port 2008.

## Contributing

Itihas is built on the idea that **small contributions add up**. You don't need to be an expert — you just need to care.

**Ways to contribute:**
- Fix a bug or improve accessibility
- Add a feature that helps communities organize better
- Improve documentation or translations
- Share your own community project ideas
- Report issues and suggest improvements

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-idea`)
3. Make your changes
4. Submit a pull request

Every contribution, no matter how small, is a micro-change that makes this platform — and the world — a little better.
